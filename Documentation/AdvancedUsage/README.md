## 何时使用高级功能

当您的应用程序需要更复杂的路由功能时，可以使用RouterKit的高级功能。例如：

- 处理外部深度链接
- 创建自定义路由匹配逻辑
- 优化大量路由的性能
- 在同一应用中使用多个路由器
- 为SwiftUI应用添加路由功能

如果您是第一次使用RouterKit，建议先阅读[快速开始](../QuickStart.md)和[核心概念](../CoreConcepts/README.md)。
