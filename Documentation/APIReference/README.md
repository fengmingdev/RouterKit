## 如何使用API参考

API参考按照类和协议组织，每个文档包含类或协议的详细说明、属性、方法和使用示例。

例如，[Router](Router.md)文档包含了Router类的所有方法和属性的详细说明，以及如何使用这些方法的示例代码。

如果您正在寻找特定功能的API，可以使用文档中的搜索功能，或者浏览上面的目录。
